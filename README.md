# 🌱 FarmGuardian AI

## Offline Agricultural Intelligence Assistant

FarmGuardian AI is a Gemma 4 powered assistant designed to help smallholder farmers identify crop problems and receive practical farming recommendations, even in areas with limited internet access.

## Current Features

- Crop diagnosis interface
- Image upload support
- Farmer problem description
- Streamlit prototype

## Future Integration

- Gemma 4 multimodal reasoning
- Crop disease detection
- Local language support
- Offline deployment
